# Mapeamento e Definição dos Tokens

Nesta seção, detalhamos todos os elementos léxicos que nosso compilador é capaz de reconhecer. O mapeamento está dividido em quatro grandes categorias para facilitar a implementação no Flex:

1. **Palavras Reservadas**: Termos protegidos da linguagem.
2. **Identificadores**: Nomes definidos pelo programador.
3. **Literais**: Valores constantes (números, strings, etc).
4. **Operadores e Pontuação**: Símbolos matemáticos e estruturais.

Selecione uma categoria no menu ao lado para ver os detalhes técnicos e tokens associados.
